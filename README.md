# Web3-Security-Data-Analytics
This repo consists of all the externship related files for Webacy's extern program for data analytics.

- Includes multiple columns for risk flags along with sources and audit links. 

### Data Analysis using Python

- Visualized using graphs for correlation analysis and frequency analysis.
